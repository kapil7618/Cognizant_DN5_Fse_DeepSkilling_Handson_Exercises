# Algorithms and Data Structures

## Overview

This repository contains implementations of fundamental **Data Structures and Algorithms** concepts using Java. The exercises focus on practical business scenarios such as inventory management, e-commerce search optimization, employee management, task management, library systems, and financial forecasting.

## Objectives

* Understand the importance of Data Structures and Algorithms in software development.
* Implement efficient data storage, retrieval, searching, sorting, and forecasting techniques.
* Analyze time complexity and optimize algorithm performance.
* Develop problem-solving skills using real-world scenarios.

## Exercises Implemented

### Exercise 1: Inventory Management System

* Product Management
* ArrayList / HashMap Operations
* Add, Update, Delete Products
* Complexity Analysis

### Exercise 2: E-Commerce Search Function

* Linear Search
* Binary Search
* Big O Analysis
* Search Performance Comparison

### Exercise 3: Sorting Customer Orders

* Bubble Sort
* Quick Sort
* Order Prioritization by Total Price
* Sorting Performance Analysis

### Exercise 4: Employee Management System

* Array Representation
* Add, Search, Traverse, Delete Operations
* Complexity Analysis

### Exercise 5: Task Management System

* Singly Linked List
* Dynamic Task Operations
* Linked List Traversal and Deletion

### Exercise 6: Library Management System

* Book Search by Title
* Linear Search
* Binary Search
* Search Algorithm Comparison

### Exercise 7: Financial Forecasting

* Recursive Algorithms
* Future Value Prediction
* Recursive Optimization Techniques

## Technologies Used

* Java
* IntelliJ IDEA / VS Code
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project in IntelliJ IDEA or VS Code.

3. Compile and run Java files:

```bash
javac Main.java
java Main
```
